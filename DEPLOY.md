# FinCalendar Frontend — Деплой на Timeweb Cloud

## Быстрый деплой

### Шаг 1: Создайте проект на Timeweb Cloud

1. Войдите в [Timeweb Cloud](https://timeweb.cloud)
2. Перейдите в **Панель управления** → **Создать проект**
3. Выберите **Node.js**
4. Подключите репозиторий: `levkinabas-star/FinCalendarWeb`
5. Выберите ветку: `frontend`

### Шаг 2: Настройки при создании проекта

```
Репозиторий:  levkinabas-star/FinCalendarWeb
Ветка:        frontend
Runtime:      Node.js 20
Build:        npm ci && npm run build
Start:        npx serve -s dist -l 3000
```

### Шаг 3: Переменные окружения (Environment)

**⚠️ ВАЖНО:** Эти переменные нужны для сборки — они вшиваются в JavaScript бандл.

| Переменная | Значение | Обязательно |
|------------|----------|-------------|
| `VITE_API_URL` | `https://api.ваш-домен.ru` | ✅ Да |
| `VITE_SUPABASE_URL` | оставить пустым | Нет |
| `VITE_SUPABASE_ANON_KEY` | оставить пустым | Нет |

### Шаг 4: Деплой

Нажмите **"Задеплоить"** — проект автоматически соберётся и запустится.

## После деплоя

1. Сайт будет доступен по адресу: `https://ваш-проект.tw1.ru`
2. Или подключите свой домен в настройках проекта

---

## Настройка API URL

После деплоя **бэкенда** обновите `VITE_API_URL` на реальный адрес вашего API.

Если адрес изменился:
1. Перейдите в настройки проекта на Timeweb Cloud
2. Измените переменную `VITE_API_URL`
3. Передеплойте проект

---

## Проверка

Откройте сайт в браузере и проверьте:
- Главная страница загружается
- Нет ошибок в консоли (F12 → Console)

---

## Оплата

Для работы оплаты:
1. Задеплойте **бэкенд** (ветка `backend`)
2. Укажите `VITE_API_URL` с адресом вашего бэкенда
3. Настройте ЮKassa webhook на адрес бэкенда
